### Bug Fixes

*   **Playback Cleanup (`PlaybackController.java`)**:
    *   Ensured that the `AudioTrack` is explicitly stopped and released when playback ends.
    *   This prevents potential resource leaks and ensures the audio hardware is available for other operations.

### Developer Tools

*   **Version Bumping Script (`bump_version.py`)**:
    *   Added a new Python tool to automate the process of bumping the application version.
    *   The script automatically updates `app/build.gradle` with the latest commit hash and increments the version number.
    *   It also automates the git staging and committing process for version updates.
